<h3 align="center">This is a Gym NoteApp application with a SQL server. It is a small app that includes CRUD, MVC, dotNET, ASP.NET, Entity Framework, LINQ.</h3>
<h4 align="center">You can creat/read/update/delete a note in the "Gym program" page. You can also search for a specific note in the search input.</h4>
<h4 align="center">You can creat/read/update/delete a workout in the "Group Workouts" page. You can also search for a specific Workout in the search input.</h4>
